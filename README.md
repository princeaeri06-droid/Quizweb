# Quizweb
 A Quiz web application built with html,css,javascript.It has a user interface and admin interface for managing the questions.

FEATURES:
1) Admin can add and delete questions.
2) Admin login had password protection.
3) Add questions stored in localstorage.
4) Get instant results just after the quiz completion.
